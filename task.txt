REST API - JSON

 
GET /users - List all users          //REST API Best practice - use this route to render HTML document on browsers
GET /api/users - List all users          //REST API Best practice - use this route to render JSON file on mobile

GET /api/users/1 - Get the user with ID 1
GET /api/users/2 - Get the user with ID 2

POST /api/users - Create new user

PATCH /api/users/id - Edit the use with ID 1

DELETE /api/users/1 - Delete the User with ID 1
